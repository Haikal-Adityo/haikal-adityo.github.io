Cinurawa Company Profile was a collaborative project I contributed to as a backend developer. My role primarily involved developing an admin panel and integrating the backend with the frontend using Laravel. For the admin interface, I chose Laravel Filament due to its user-friendly design and efficiency in handling basic CRUD operations.

  • Created responsive and efficient admin panel using Laravel Filament.
  • Implemented backend logic for seamless interaction between admin and frontend.
  • Ensured data security and integrity through proper validation and authentication mechanisms.
  • Optimized database queries for enhanced performance.
  • Collaborated with frontend developers to integrate backend functionalities seamlessly.
